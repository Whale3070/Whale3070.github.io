SMART原则（具体、可以量化、可以实现、与公司目标&职责相关、有时间限制）

OKR与KPI的区别

KPI 指的是客户满意度

  1. OKR是一种管理方法，KPI是一种考核工具

​ 前者与考核、薪酬（绩效）无关，后者有关

  1. OKR是员工定义、与上级沟通确定的

    KPI是由上至下层层分派的

  2. OKR指定的目标富有挑战性，可以容忍失败

    KPI要求百分百完成

良好的目标 Object

  1. 必须是可以达成的，不能不切实际
  2. 本季度可以执行
  3. 结果可以控制
  4. 对业务有价值

KR - key result 关键结果

  1. 实现O的不同策略/路径
  2. 任务型OKR
  3. 价值型OKR

OKR使用

  1. 编写OKR，确定
  2. 与同事对齐
  3. @相关同事
  4. 进度更新（1一周更新一次）
  5. KR填写2 到 5条
  6. 设置查看权限
